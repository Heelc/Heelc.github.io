按钮

在HTML中按钮有3中类型：

  1. Submit 提交表单数据到服务器
  2. Reset 重置表单控件到默认值
  3. Button 无实际作用按钮，可通过js定制特殊功能（非常常用）
    Submit

Rest

Button

使用<input>或者<button>来创建按钮都有相同的行为能力，但是他们有显著的不同之处:
<input>到底还是个空元素，它不能插入HTML内容，只能插入到value属性中，因此只能接受纯文本内容。
<button>可以插入HTML，在它的闭合标签前，它可以被相应地样式化

高级表单部件

数字
<input type="number" name="age" id="age" value="10" max="10" min="1">这将创建一个数字部件，默认step为1，范围限制到1~10，点击按钮数值会增加或减少1

滑块
滑块是为了选择一个确切值不那么重要的数值。还是将万能的input-type设置为range来创建滑块，配置max、mix、step基本上是必不可少的。

为了使滑块的改变变得显而易见，我们用js来实现视觉反馈

日期选择器
对于web开发来说，对日期和时间值一直是一件头疼的事。HTML5对此有了些许改进，减轻了开发人员的工作量

其他小部件

一些有用的小部件

文件选择器
请求图像文件，允许多选文件

隐藏内容

图像按钮
拥有两种元素的属性，
<input type="image" name="imageButton" alt="Click me!" src="my-img.png" width="80" height="30">
